## §12.8 季度仪表盘（QBR Dashboard）

每季度董事会报告必含 9 项 KPI：

```
┌─────────────────────────────────────────────────────────┐
│   InvestMind QBR  ·  季度 X / 年度 Y                    │
├─────────────────────────────────────────────────────────┤
│  1. 付费用户期末     │ █████████░░  74% YoY  ▲        │
│  2. ARR 期末        │ █████████░░  82% YoY  ▲        │
│  3. NRR             │ █████████░░  115%      ▲        │
│  4. LTV/CAC         │ █████████░░  3.4×       ▲        │
│  5. Payback (mo)    │ █████░░░░░░  11 mo    ▼ (好)    │
│  6. Magic Number    │ ████████░░░  1.25       ▲        │
│  7. 评分 AUC         │ ██████░░░░░  0.81       ▲        │
│  8. NPS             │ █████████░░  +66        ▲        │
│  9. 现金 Runway     │ ████████░░░  18 mo     —         │
└─────────────────────────────────────────────────────────┘
```

每季度 BP（Business Plan）会议会基于这些 KPI 做 +1/-1 调整：

- 任何 ↗ 5%+ 的指标得到加倍投入
- 任何 ↘ 10%+ 的指标进入"重点关注"清单
- 连续 2 季度落后基线 → 启动负责人轮换
