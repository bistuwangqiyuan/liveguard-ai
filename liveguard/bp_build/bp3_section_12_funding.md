# §12 天使-优先融资计划与 Cap Table 回报

> 来源：`11_fundraising_dilution.py`（Cap Table 全稀释）、`18_angel_returns.py`（天使条件回报）、
> `14_monte_carlo_valuation.py`（退出锚点）。本章以**天使投资人 5 年回报**为中心组织。

## Key Takeaways

1. **5 轮融资以天使轮为起点**：Angel → Seed → A → B → C，**累计融资 ¥20.6 亿**，C 轮 Post-money ¥120 亿。
2. **天使全稀释路径**：入场 20% → C 轮后 **8.84%**；创始团队 C 轮后保留 **35.4%**，控制权稳固（配合 AB 股）。
3. **天使 5 年纸面 mark**（按 C 轮 Post ¥120 亿）：MOIC **106×** / IRR **154%**——即便不退出，账面价值已极高。
4. **条件于成功退出**：保守 ¥130 亿 → 115×；中性 ¥222 亿 → 196×；乐观 ¥352 亿 → 311×；极乐观 ¥500 亿 → 442×。
5. **诚实口径**：上述为条件于成功（概率 7.1%）；**概率加权期望 MOIC ≈ 19.7×、IRR ≈ 82%** 见 §11。

## §12.4 天使 5 年回报（条件于成功退出）

> 天使 ¥1,000 万，全稀释后 8.84%。5 年退出回报 = 退出 EV × 8.84%。

| 退出情景 | 退出 EV (亿)| 天使回报 (亿)| MOIC | IRR (5 年)|
|---------|------------:|------------:|-----:|---------:|
| 保守（战略并购）| 130 | 11.5 | **115×** | 158% |
| 中性（加权综合 EV）| 222 | 19.7 | **196×** | 188% |
| 乐观（IPO, MC P90）| 352 | 31.1 | **311×** | 215% |
| 极乐观（头部 IPO）| 500 | 44.2 | **442×** | 238% |

> **5 年纸面 mark**：即便不退出，按 C 轮 Post ¥120 亿计，天使账面价值 ¥10.6 亿 → MOIC **106×** / IRR **154%**。

> ⚠️ **口径提醒**：上表为**条件于公司成功走到退出**（概率 7.1%，见 §11）。
> 纳入失败概率后的**概率加权期望** = MOIC ≈ 19.7×、5 年 IRR ≈ 82%（§11.2）。两者一同呈现，方为完整的天使风险-收益画像。
