# §8 财务预测与单位经济

> 本章所有数字来自 `10_financial_projections.py` 的 5 年三大报表与 `15_sensitivity_analysis.py`。
> 复现：`cd liveguard/models/python_models && python run_all.py`。
> **资产负债表勾稽差异 = 0.00 元（精确成立）。**

## Key Takeaways

1. **5 年收入路径**：Y1 0.13 亿 → Y2 1.04 亿 → Y3 4.47 亿 → Y4 12.46 亿 → **Y5 27.95 亿元**。
2. **Y5 净利 3.88 亿元、净利率 14%**；毛利率由 Y1 62% 提升至稳态 78%。
3. **Y5 Rule of 40 = 142**（增速 124% + 营业利润率 18%）；Magic Number 由 1.16 升至 1.73；Burn Multiple Y5 = 0。
4. **现金路径稳健**：Y3 末现金 3.41 亿，Y4 末最紧（0.67 亿），Y5 末 9.93 亿元。
5. **关键敏感性**：Enterprise 客单价与占比是收入第一驱动；退出倍数与 Y10 EBITDA 是估值第一驱动。
