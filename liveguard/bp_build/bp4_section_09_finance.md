# §9 财务预测（5 年三大报表）

> 全部数字来自 `10_financial_projections.py`（收入口径=四层货币化总收入 `20`）。
> 复现：`cd liveguard/models/python_models && python run_all.py`。
> **资产负债表勾稽差异 = 0.00 元（精确成立）。**

## Key Takeaways

1. **5 年收入路径**：0.13 → 1.21 → 5.99 → 18.56 → **45.27 亿元**（核心 SaaS 27.95 亿 + 扩展层 17.33 亿）。
2. **Y5 EBITDA 10.20 亿、净利 6.99 亿、净利率 15%**；混合毛利率由 78% 抬升至 **80%**。
3. **Y5 Rule of 40 = 164**（增速 144% + 营业利润率 20%）；Magic Number 1.18→1.84；Burn Multiple Y5 = 0。
4. **现金路径稳健**：Y3 末 6.25 亿，Y4 末最紧（2.38 亿，仍为正），Y5 末 14.86 亿。
5. **关键敏感性**：Enterprise 客单价/占比是核心收入主驱动；退出倍数与 Y10 EBITDA 是估值主驱动（§10）。
