# §14 团队与治理结构

## Key Takeaways

1. **核心团队 7 人**：均有 CV / 大型 SaaS / 直播电商 10+ 年深度经验，覆盖"算法-工程-产品-商业-合规"。
2. **顾问委员会 6 人**：学术、行业、合规、资本全要素。
3. **5 年组织**：35 → 480 人，研发占比 57%（Y1）→ 38%（Y5）。
4. **治理"三会一委"** + AB 股结构：王启源合计持股 **35.37%**（Founders 28.3% + PreAngel 7.1%）→ 投票权绝对控制。

## §14.4 公司治理

- **AB 股双层结构**：创始团队 A 类股（1 股 = 10 票），机构 B 类股（1 股 = 1 票）；C 轮后创始团队持股 35.4% → 投票权 > 78%。
- **董事会（Y3 后）**：9 席（3 创始人 + 4 投资人 + 2 独立董事）；委员会：审计 / 薪酬 / 提名 / AI 伦理与合规。
- **天使投资人权益**：标准优先股 + 信息权 + 按比例反稀释 + 优先清算 1×（§12.6）。
- **三道防线**：业务部门 / 合规法务 / 内审 + 外部审计。
