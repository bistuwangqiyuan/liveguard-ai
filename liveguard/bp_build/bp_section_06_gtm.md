# §6 市场进入与营销战略（GTM）

> 本章基于 §3 竞争定位与 §5 商业模式，给出 5 年 GTM 路线图、Channel-Mix、Funnel 设计、获客成本目标。
> 客户数、CAC、S&M 预算均与 `07_growth_cohort.py` / `10_financial_projections.py` 一致。

## Key Takeaways

1. **三阶段 GTM**：Y1–Y2 PLG（产品自驱长尾 SMB）→ Y2–Y3 SLG（销售驱动中型 MCN）→ Y3+ ABM（大客户 + 平台合作）。
2. **8 大获客渠道矩阵**：内容/SEO/社群（PLG）+ 直销/代理/合作伙伴（SLG）+ 战略合作/MCN 联盟（ABM）。
3. **Funnel 转化基线**：MQL→SQL 30% / SQL→Win 22%；销售周期 SMB 14 天 / 中型 65 天 / KA 180 天。
4. **5 年获客曲线**：Y1 800 → Y2 6,500 → Y3 28,000 → Y4 78,000 → Y5 175,000；S&M/收入由 90% 降至 32%。

## §6.4 PLG 自助转化（Phase 1 核心）

```
注册 → 选直播平台 → 一键授权 → 自动监控 5 个直播间 → 模拟告警体验
  → 首次告警送达(T+1h) → 客户成功 1-on-1 onboarding(3 天)
  → 试用回顾报告 + 转化券(7 天) → 到期前 3 天提醒 + 限时折扣
```

内容矩阵：行业白皮书（季度）、技术博客（周 2 篇）、短视频教程（周 3 条）、MCN 案例（月 4 篇）、《中国直播质量年度报告》IP 化（月 1 篇）。

## §6.6 ABM — 大客户与平台合作

每个 KA 配"三人小组"：1 KA AE + 1 解决方案架构师 + 1 客户成功经理。KA 六步法：目标识别 → 多触点接触 → PoC 设计 → 方案定制 → 采购陪跑 → 续约扩展。

**平台战略合作（最重要护城河之一）**：
- 模式 1（首选）：官方合作伙伴 + SDK 嵌入主播工作台 → 平台分润 15–20%
- 模式 2：API 集成，按调用计费
- 模式 3：联合发布行业标准/白皮书
- Y2 H2 签首家平台 PoC，Y3 H1 正式发布。

## §6.7 GTM 预算（与 §8 财务一致）

| 年度 | S&M 预算（万元）| 占收入 | 主要分配 |
