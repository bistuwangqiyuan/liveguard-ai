# §12 发展路线图与里程碑

## Key Takeaways

1. **5 年三阶段战略**：Y1–Y2 立基（PMF + 算法领先）→ Y3–Y4 规模（全国头部 + 平台合作）→ Y5+ 平台化（多产品 + 国际化 + IPO）。
2. **22 个 SMART 里程碑**，每一个都有可量化触发指标 + 时间窗 + 责任人。
3. **季度仪表盘**治理：30+ 关键 KPI 月度跟踪。
4. **触发式融资节奏**：里程碑达成 → 触发下一轮融资启动。

---

## §12.1 三阶段战略路径

```
            产品形态                  GTM 重心                 组织规模
 Y1 ─── 单产品 SaaS（MVP→1.0）    SMB + 早期 MCN          35 人
 Y2 ─── 单产品 + 边缘设备          头部 MCN 突破            105 人
 Y3 ─── 产品矩阵 v1（监控+审核）   平台合作 + KA            210 人
 Y4 ─── 产品矩阵 v2（多场景）      行业拓展 + 海外试点      350 人
 Y5+ ── 平台化（PaaS + 开发者）    IPO 准备 + 海外正式       480 人
```

## §12.3 季度仪表盘（QBR Dashboard，节选 30 KPI）

- **财务（10）**：ARR、MRR、Booking、Revenue、毛利率、Burn、Cash、CAC、Payback、LTV/CAC
- **客户（8）**：Logo 总数、新签、流失、ARPU、NRR、GRR、扩展收入比、Logo 留存
- **产品（6）**：DAU/MAU、活跃账户、API 调用量、NPS、新功能采纳率、Bug 关闭率
- **技术（4）**：FAR、FNR、P95 延迟、单位推理成本
- **组织（2）**：关键岗位空缺率、核心人才保留率
