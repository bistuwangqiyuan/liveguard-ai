# §15 发展路线图与里程碑

## Key Takeaways

1. **5 年三阶段**：Y1–Y2 立基（PMF + 算法领先 + 风控 OS）→ Y3–Y4 规模（平台合作 + 可信网络 + 保险）→ Y5+ 平台化（PaaS + 国际化 + IPO）。
2. **四层货币化按年解锁**：监控（Y1）→ 风控 OS（Y2）→ 数据网络（Y3）→ 保险/RegTech（Y4–Y5）。
3. **触发式融资**：里程碑达成 → 触发下一轮（§8.6 22 个 SMART 里程碑）。


## §15.3 季度仪表盘（QBR，30+ KPI 节选）

- **财务（10）**：ARR、MRR、Booking、Revenue、毛利率、Burn、Cash、CAC、Payback、LTV/CAC
- **客户（8）**：Logo、新签、流失、ARPU、NRR、GRR、扩展收入比、Logo 留存
- **产品（6）**：DAU/MAU、活跃账户、API 调用量、NPS、采纳率、Bug 关闭率
- **技术（4）**：FAR、FNR、P95 延迟、单位推理成本
- **组织（2）**：关键岗位空缺率、核心人才保留率
