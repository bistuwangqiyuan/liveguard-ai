# §1 公司与产品概述

## Key Takeaways

1. **公司定位**："直播经济的 AI 安全官"，专注为直播平台、MCN、品牌商家、行业客户提供 24×7 直播监控智能体 SaaS。
2. **使命愿景**：让每一场直播都安全、合规、可信。10 年内成为全球直播 AI 监控领域的事实标准。
3. **三大产品线（Y3 末）**：① 直播监控智能体（核心）② 主播身份验证 API ③ 行业垂直解决方案（教培/政务/医疗）。
4. **首次披露指标（Y1）**：MVP 上线 → 30 试用 → 800 付费账号 → ARR 1,277 万元。

## §1.2 使命、愿景与价值观

### §1.2.1 使命（Mission）
> **让每一场直播都安全、合规、可信。**

直播已经成为中国 8.16 亿用户日常生活的一部分，但人工监控成本高、漏报多，平台与商家承担巨大的合规与运营风险。我们用 AI 赋能直播经济的"安全基础设施"。

### §1.2.2 愿景（Vision）
> **10 年内成为全球直播 AI 监控领域的事实标准——任何一场直播，背后都有"守播"在保驾护航。**

### §1.2.3 价值观（Values）
- **客户成功为先**（Customer First）
- **极致工程师文化**（Engineering Excellence）
- **诚信合规**（Integrity & Compliance）
- **长期主义**（Long-Term Orientation）

## §1.3 核心产品（Y1 — 直播监控智能体）

### §1.3.1 一句话定义
> "把摄像头对准直播间，AI 实时判断主播是否在岗、是否本人、是否合规——离岗 60 秒内多通道告警。"

### §1.3.2 核心功能
1. **主播离岗检测**：60 秒内识别"主播不在画面 / 离开镜头"并告警
2. **主播身份持续验证**：识别"替身""代播"等身份切换
3. **多通道告警**：电话 / 短信 / 钉钉 / 飞书 / APP 推送
4. **多平台支持**：抖音 / 淘宝 / 快手 / 视频号 / TikTok 等主流平台
5. **告警闭环**：事件录像截取 + 自动归档 + 一键复盘
6. **可选模块**：行为合规 / 话术合规 / 数据看板（需额外购买）

### §1.3.3 三大形态
- **SaaS Web 控制台**（主形态）
- **手机 APP**（轻量管理 + 接收告警）
- **边缘智能盒子**（私有化 / 隐私敏感场景）
