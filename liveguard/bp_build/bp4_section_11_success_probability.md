# §11b 创始人王启源个人决策（胜率 × 盈亏比 × 凯利）

> 全部来自 `21_personal_kelly_founder.py`。经济账户：**500 万现金 + 5×100 万时间成本 = 1000 万 bankroll**；投入强度 **15–20 小时/周**；闸门 **STAGE_GATES_FOUNDER**（每级 p_advance ×0.82，[S-042]）。

## Key Takeaways

1. **胜率 p = 27.9%**（P(净回报>0)）；**盈亏比 b = 129.3**——赢局期望远大于输局，但赢局概率低。
2. **Kelly 全量 f\* = 27.4%**；**建议仓位（1/4 Kelly）= 6.8% ≈ ¥68 万**——勿将 1000 万全投入误读为最优。
3. **P(成功退出) = 2.6%**（天使基准 7.1%）；**P(全损) = 72.1%**；**中位 MOIC = 0**。
4. **行动标签**：轻仓试探 + 必须补全职技术负责人。
5. **期望值≠建议仓位**：全投入期望净回报约 +¥3.54 亿，由 2.6% 成功路径的极端右尾拉动；凯利约束的是**个人财富风险暴露**。

## §11b.2 胜率、盈亏比与凯利公式

| 符号 | 定义 | 模型值 |
|------|------|------:|
| **p（胜率）** | P(净回报 > 0) | **27.9%** |
| **b（盈亏比）** | E[净回报\|赢] / \|E[净回报\|输]\| | **129.3** |
| **f\*** | max(0, (p(b+1)−1)/b) | **27.4%** |
| **建议 f** | min(f\*×0.25, 35%, 现金上限50%) | **6.8%** |

```
f* = (p × (b + 1) − 1) / b = (0.279 × 130.3 − 1) / 129.3 ≈ 0.274
建议仓位 = 0.25 × f* ≈ 6.8%  →  总承诺 ≈ ¥68.4 万（现金约 ¥34.2 万 + 时间等价 ¥34.2 万）
```

> **为何全量 f\* 27% 却只建议 6.8%？** 四分之一凯利用于吸收参数误差与高方差；硬上限 35% 防止单笔创业赌注过重。

## §11b.4 敏感性（课余折扣系数）

| 折扣乘数 | P(成功退出) |
|---------|------------:|
| 0.78（更重折扣）| 更低 |
| **0.82（基线）** | **2.6%** |
| 0.86（略乐观）| 更高 |

> 提升 p 的最有效杠杆：**全职化技术负责人**或 **降低个人时间约束**——否则模型不允许在 BP 中使用全职创始的晋级率。
