# §9 融资计划与估值

> 来源：`11_fundraising_dilution.py`、`12_valuation_dcf.py`、`13_valuation_comparables.py`、
> `14_monte_carlo_valuation.py`。所有数字可一键复现。

## Key Takeaways

1. **5 轮融资 + ESOP 总规划**：Seed → Pre-A → A → B → C，**累计融资 14.93 亿元**，C 轮 Post-money **80 亿元**。
2. **创始团队 C 轮后保留 33.7% 股权**，ESOP 9.0%，机构合计 57.2%——结构稳健，控制权明确。
3. **多模型估值收敛**：DCF（两阶段+18×）149 亿 / Gordon 对照 58 亿 / EV/Sales 中位 129 亿 / EV/EBITDA 中位 143 亿 / 蒙特卡洛 P50 148 亿 → **加权综合 EV ≈ 135 亿元**。
4. **退出路径清晰**：B/C 轮战略并购（字节/阿里/腾讯/360/华为云）或 Y6–Y7 科创板 / 港股 18C IPO。

## §9.4 估值方法学（多模型加权）

### §9.4.1 两阶段 DCF + 退出倍数

> 来源：`12_valuation_dcf.py`。显性期 Y1–Y5 取模型 FCF；过渡期 Y6–Y10 增速递减 30%→18%、营业利润率 20%→28%；终值 Y10 退出 EV/EBITDA = 18×；WACC = 14%。

| 项目 | 值（亿元）|
|------|----------:|
| PV(显性 Y1–Y5 FCF) | -3.5 |
| PV(过渡 Y6–Y10 FCF) | 17.9 |
| PV(终值 @18× EBITDA Y10) | 134.9 |
| **EV（两阶段 + 退出倍数）** | **149.3** |
| EV (Gordon 永续 g=3% 对照) | 57.8 |

![两阶段 DCF 价值桥与 FCF 曲线](liveguard/docs/images/charts/fig_12_valuation_dcf.png)

> Gordon 模型给出的 58 亿是**保守下限**——它假设 Y5 后永远以 3% 增长，显著低估高增长 SaaS。两个模型组合给出 58–149 亿合理区间。

### §9.4.2 可比公司估值

> 来源：`13_valuation_comparables.py`。12 家可比池：商汤、旷视、依图、Verint、NICE、Asana、Monday、Sprinklr、微盟、有赞、Salesforce、Datadog。

| 倍数 | P25 | 中位 | P75 | 应用于 Y5 → EV (亿)|
|------|----:|----:|----:|-------------------:|
| EV/Sales（Y5 收入 27.95 亿）| 3.4× | 4.6× | 6.6× | 95 / **129** / 184 |
| EV/EBITDA（Y5 EBITDA 5.74 亿）| 18× | 25× | 36× | 103 / **143** / 207 |

![可比公司估值](liveguard/docs/images/charts/fig_13_valuation_comparables.png)

### §9.4.3 蒙特卡洛模拟（N=200,000）

> 来源：`14_monte_carlo_valuation.py`。EV = 0.5×简化 DCF + 0.5×EV/Sales。

| 分位数 | EV (亿元)| | 分位数 | EV (亿元)|
|-------:|---------:|--|-------:|---------:|
| P5 | 84 | | P75 | 183 |
| P10 | 96 | | P90 | 217 |
| P25 | 119 | | P95 | 239 |
| **P50** | **148** | | 均值 | 153 |

### §9.4.4 综合估值

| 方法 | 权重 | EV (亿元)| 加权贡献 |
|------|----:|---------:|--------:|
| DCF（两阶段 + 退出）| 25% | 149.3 | 37.3 |
| DCF（Gordon 对照）| 10% | 57.8 | 5.8 |
| EV/Sales（中位）| 20% | 129 | 25.8 |
| EV/EBITDA（中位）| 15% | 143 | 21.5 |
| 蒙特卡洛（P50）| 30% | 148.4 | 44.5 |
| **加权综合 EV** | 100% | — | **≈ 135 亿元** |

![多模型加权估值](liveguard/docs/images/charts/fig_14_valuation_montecarlo.png)

> 投资人对话区间：**110 – 160 亿元 RMB**。C 轮 Post-money 80 亿（综合估值的 P25–P50 之间）为"留有上行空间"的友好估值。

## §9.5 退出路径与 A 轮投资人回报

> A 轮投资人 1.0 亿 @ Pre 5.0 亿，C 轮后稀释至 **11.7%**（`11_fundraising_dilution.py`）。

| 退出情景 | 退出 EV (亿)| 退出回报 (亿)| MOIC | IRR (5 年)|
|---------|------------:|------------:|-----:|---------:|
| 保守（M&A）| 100 | 11.7 | **11.7×** | 64% |
| 中性（M&A / 早期 IPO）| 180 | 21.0 | **21.0×** | 84% |
