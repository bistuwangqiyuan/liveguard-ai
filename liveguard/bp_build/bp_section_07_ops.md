# §7 运营与组织计划

> 本章描述端到端运营体系：客户成功、技术运维（SRE/NOC）、供应链（边缘硬件）、合规运营、组织建设。
> NOC 容量与告警抑制策略由 `05_alerts_capacity_erlangc.py` / `06_dedup_suppression_sim.py` 量化。

## Key Takeaways

1. **NOC 7×24 监控值班**：Erlang-C 模型测算——10,000 路并发仅需 **36 名坐席**即可达成 90% 告警 60 秒内响应；规模化坐席需求呈亚线性增长。
2. **告警去重/抑制**：推荐 W=300s 自适应策略，平衡人工降载与漏报；P0 告警不进入抑制窗口直通。
3. **三层客户成功体系**：Onboarding → Adoption → Renewal & Expansion；目标 NRR ≥ 125%、NPS ≥ 60。
4. **边缘硬件供应链**：与 4 家 ODM 战略合作（地平线、瑞芯微、寒武纪、华为昇腾），月产能弹性 500–5,000 台。

## §7.3 数据与算法运营

### §7.3.1 数据飞轮闭环（每周迭代）

```
客户授权 → 边缘端难例采样（仅特征向量）→ 匿名化样本池
       → 主动学习 + 人工标注（质检 100%）→ 模型微调
       → A/B 测试 → 灰度发布 → 全量 OTA 更新
```

- 数据增长：Y1 5 万小时 / Y3 50 万 / Y5 500 万小时
- 标注团队：30 人专属 + 200 人外包；4 道质检
- 模型迭代：核心模型周更，专项垂直模型月更；5 分钟内可一键回滚

## §7.5 合规与安全运营

- **合规闭环**：季度法律风险评审（CCO + 外部律所）；月度数据治理审计（DPO）；年度 ISO 27001 / SOC 2 续审；政策预警 24 小时同步。
- **信息安全**：三防一体（WAF/DDoS + 零信任 + 加密/DLP）；季度红队渗透；年度等保三级测评。
- **数据生命周期**：采集（告知+同意+最小化）→ 传输（TLS 1.3 + 国密）→ 存储（分级加密、冷热隔离、多副本灾备）→ 使用（访问审计）→ 销毁（离网 30 天内彻底删除）。
