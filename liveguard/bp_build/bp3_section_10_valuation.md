# §10 估值（多模型加权）

> 来源：`12_valuation_dcf.py`、`13_valuation_comparables.py`、`14_monte_carlo_valuation.py`。
> 所有数字一键复现。估值用于 §12 天使回报与 §11 成功概率的退出锚点。

## Key Takeaways

1. **多模型收敛**：DCF 两阶段 ¥243 亿 / Gordon 对照 ¥95 亿 / EV/Sales 中位 ¥208 亿 / EV/EBITDA 中位 ¥255 亿 / 蒙特卡洛 P50 ¥240 亿 → **加权综合 EV ≈ ¥222 亿元**。
2. **C 轮 Post ¥120 亿**显著低于加权综合 EV（¥222 亿），为投资人留有充足上行空间。
3. **退出路径清晰**：Y4–Y5 战略并购（字节/阿里/腾讯/360/华为云）¥130–250 亿；Y6–Y7 科创板/港股 18C/NASDAQ ADR，目标 ¥250–500 亿。

## §10.1 两阶段 DCF + 退出倍数（来源 `12`）

> 显性期 Y1–Y5 取模型 FCF；过渡期 Y6–Y10 增速 30%→18%、营业利润率 20%→28%；终值 Y10 退出 EV/EBITDA = 18×；WACC = 14%。

| 项目 | 值（亿元）|
|------|----------:|
| PV(显性 Y1–Y5 FCF) | -4.1 |
| PV(过渡 Y6–Y10 FCF) | 29.0 |
| PV(终值 @18× EBITDA Y10) | 218.5 |
| **EV（两阶段 + 退出倍数）** | **243.4** |
| EV (Gordon 永续 g=3% 对照) | 95.1 |

![两阶段 DCF 价值桥与 FCF 曲线](liveguard/docs/images/charts/fig_12_valuation_dcf.png)

> Gordon 模型 ¥95 亿是保守下限（假设 Y10 后永远 3% 增长，低估高增长 SaaS）。两模型组合给出 ¥95–243 亿合理区间。

## §10.4 综合估值（多模型加权）

| 方法 | 权重 | EV (亿元)| 加权贡献 |
|------|----:|---------:|--------:|
| DCF（两阶段 + 退出）| 25% | 243.4 | 60.9 |
| DCF（Gordon 对照）| 10% | 95.1 | 9.5 |
| EV/Sales（中位）| 20% | 208 | 41.6 |
| EV/EBITDA（中位）| 15% | 255 | 38.3 |
| 蒙特卡洛（P50）| 30% | 240.4 | 72.1 |
| **加权综合 EV** | 100% | — | **≈ 222 亿元** |

![多模型加权估值](liveguard/docs/images/charts/fig_14_valuation_montecarlo.png)

> 投资人对话区间：**¥150 – 260 亿元**。C 轮 Post ¥120 亿（综合估值的 P10–P25 区间）为"留足上行空间"的友好估值。
