# §11 项目成功概率与预期投资收益（天使视角）

> 本章是 v3.0 的核心交付物之一，直接回答天使投资人最关心的两个问题：
> **这个项目成功的概率有多大？我的预期收益是多少？**
> 全部来自 `19_success_probability.py`（阶段闸门生存模型 + 蒙特卡洛 N=200,000，seed=42）。

## Key Takeaways

1. **P(成功大额退出) ≈ 7.1%**——天使 → Seed → A → B → C → 退出，逐级前进概率相乘（对标早期 AI SaaS 行业基准 [S-040][S-041]）。
2. **概率加权期望：MOIC ≈ 19.7×、5 年 IRR ≈ 82%**——这是把失败计入后的"诚实"期望收益。
3. **诚实披露下行**：P(本金全损) ≈ 69.5%，中位结局 = 本金全损；期望由右尾（7.1% 概率的 217× 成功退出）驱动——这是典型的**高方差、右尾驱动**天使资产。
4. **收益分布**：P(不亏钱 MOIC≥1×)=30.5%，P(≥3×)=25.0%，P(≥10×)=16.9%，P95 MOIC≈153×。
5. **稳健性**：即使关键阶段晋级概率 −10pp，期望 IRR 仍 ≥ 75%。

## §11.1 阶段闸门生存模型

> 早期公司的"成功"是一连串闸门的连乘。每个闸门有"前进/部分退出(并购)/归零"三种结局。

| 闸门 | 时点 | 前进概率 | 累计到达概率 |
|------|-----:|--------:|------------:|
| 天使 → Seed | Y1 H2 | 72% | 72.0% |
| Seed → A | Y2 | 55% | 39.6% |
| A → B | Y3 | 50% | 19.8% |
| B → C | Y3 末 | 55% | 10.9% |
| C → **成功退出** | Y5 | 65% | **7.1%** |

```
P(成功退出) = 0.72 × 0.55 × 0.50 × 0.55 × 0.65 = 7.1%
```

> 概率取值依据：早期 AI/SaaS 种子→A 晋级率 ~35–55%、A→B ~45–55%、成长期存活率更高（CB Insights / 红杉 / Correlation Ventures 基准 [S-040][S-041]），并按本项目优势（监管刚需、强团队、数据护城河）做正向微调。

## §11.2 天使预期投资收益（概率加权）

> 蒙特卡洛对每条路径模拟结局（归零 / 部分退出 / 成功退出），按全稀释股比与退出估值计算天使回报，汇总期望。

| 指标 | 数值 |
|------|-----:|
| **期望 MOIC（概率加权）** | **≈ 19.7×** |
| **期望 5 年 IRR** | **≈ 82%**（= 期望 MOIC^(1/5) − 1）|
| 中位 MOIC | 0.00×（中位结局 = 本金全损）|
| P(本金全损) | 69.5% |
| P(部分退出，acqui-hire/小额并购) | 23.5% |
| P(成功大额退出) | 7.1% |
| 条件于成功：MOIC / IRR | ≈ 217× / ≈ 193% |

### MOIC 分布分位（含全损样本）

| 分位 | MOIC |
|-----:|-----:|
| P50 | 0.0× |
| P75 | 1.2× |
| P90 | 58.9× |
| P95 | 153.0× |
| P99 | 320.0× |

> **如何理解 82% 的期望 IRR**：天使是单笔高方差投资，约 69.5% 的路径本金全损，但 7.1% 的成功路径回报达 ~217×。
> 期望值由右尾驱动——这正是顶级天使资产的特征（"幂律分布"）。我们不掩饰高失败率，而是用可复现模型把它量化清楚。

## §11.3 收益命中率（投资人决策口径）

| 事件 | 概率 |
|------|-----:|
| 不亏钱（MOIC ≥ 1×）| 30.5% |
| 翻 3 倍以上（≥ 3×）| 25.0% |
| 翻 10 倍以上（≥ 10×）| 16.9% |
| 翻 100 倍以上（≥ 100×）| ~6% |

> 对天使组合而言：单笔 ¥1,000 万、期望 19.7× → 期望回报 ¥1.97 亿。一个 8–10 笔同类资产的组合，
> 在幂律下大概率由 1 笔决定整体回报——本项目正是为"成为那 1 笔"而设计。

## §11.4 期望 IRR 的稳健性（对阶段闸门 ±10pp 敏感性）

| 关键闸门 | 前进概率 −10pp | 基线 82% | 前进概率 +10pp |
|---------|--------------:|--------:|--------------:|
| A → B | 75% | 82% | 87% |
| B → C | 76% | 82% | 86% |
| C → 成功退出 | 77% | 82% | 85% |

> 即使最关键的成长期闸门概率下调 10 个百分点，天使期望 5 年 IRR 仍 ≥ 75%——结论稳健。
