# §6 市场进入与营销战略（GTM）

> 基于 §3 竞争定位与 §5 商业模式，给出 5 年 GTM 路线图、Channel-Mix、Funnel、获客成本目标。
> 客户数、CAC、S&M 预算均与 `07_growth_cohort.py` / `10_financial_projections.py` 一致。

## Key Takeaways

1. **三阶段 GTM**：Y1–Y2 PLG（长尾 SMB 自驱）→ Y2–Y3 SLG（销售驱动中型 MCN）→ Y3+ ABM（大客户 + 平台合作 + 数据网络）。
2. **land-and-expand 是 v3 核心增长引擎**：以监控低价切入 → 加购风控 OS → 接入数据网络/保险，单客户价值随时间增长（NRR 97%→145%）。
3. **Funnel 基线**：MQL→SQL 30% / SQL→Win 22%；销售周期 SMB 14 天 / 中型 65 天 / KA 180 天 / 平台 360 天。
4. **S&M/收入由 90% 降至 32%**，Magic Number 由 1.18 升至 1.84。

## §6.1 GTM 三阶段路线图

| 阶段 | 时点 | 核心策略 | 目标客户 |
|------|------|---------|---------|
| **PLG** | Y1–Y2 H1 | 14 天免费试用 + 内容营销 + 插件零接入 | SMB 主播、初创 MCN |

## §6.4 平台战略合作（最重要护城河之一）

- **模式 1（首选）**：官方合作伙伴 + SDK 嵌入主播工作台 → 平台分润 15–20%。
- **模式 2**：API 集成，按调用计费（接入数据网络/可信认证）。
- **模式 3**：联合发布行业标准/白皮书 + 保险公司联合承保"合规直播"。
- Y2 H2 签首家平台 PoC，Y3 H1 正式发布；Y4 接入保险合作。

| 年度 | S&M 预算（万元）| 占收入 | 主要分配 |
